# ChinaPak ImportHub Documentation

This folder is reserved for product, architecture, and implementation notes.

Source of truth:

- `MDOM.md` is the primary business and product source of truth.
- `PRD-v0.1.pdf` is provided for human reading only.
- If `AGENTS.md` and `MDOM.md` conflict, follow `AGENTS.md` and flag the conflict in the task summary.

Current foundation:

- Next.js App Router application structure.
- Urdu-first importer-facing public routes.
- Chinese-first FMS/factory route preparation.
- English admin route preparation.
- Shared brand configuration for colors, navigation, packages, and official route constants.

Intentional placeholders:

- Supabase/PostgreSQL auth and database are not connected.
- Payments are not implemented.
- Messaging is not implemented.
- Dashboards and private portals are reserved routes only.
- Factory portal exists as architecture but remains inactive and unlinked.
